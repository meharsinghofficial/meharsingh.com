# Mehar Singh

Minimal personal portfolio site for `meharsingh.com`.

Upload these files directly to the root of the GitHub repository connected to Vercel:

- `index.html`
- `styles.css`
- `mehar.jpg`  ← profile photo (required by the page)
- `CNAME`
- `Mehar_Singh_Resume.pdf`

The site is static and needs no JavaScript. Do not upload a zip file to the repository.
