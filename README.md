# Mehar Singh Portfolio Site

This is a static portfolio website built from Mehar Singh's resume.

## Files

- `index.html` - main website page
- `styles.css` - responsive styling
- `script.js` - mobile navigation and current year
- `assets/Mehar_Singh_Resume.pdf` - downloadable resume

## Deploy

Upload the contents of this folder to any static host, such as Netlify, Vercel, GitHub Pages, Cloudflare Pages, or your domain provider's file hosting.

The home page is `index.html`.

## Custom domain

This folder includes a `CNAME` file for:

```txt
meharsingh.com
```

If you use GitHub Pages, keep the `CNAME` file in the published folder and add `meharsingh.com` as the custom domain in the repository Pages settings.

For Netlify, Vercel, or Cloudflare Pages, add `meharsingh.com` in the hosting dashboard, then copy the DNS records they provide into your domain registrar's DNS settings.
